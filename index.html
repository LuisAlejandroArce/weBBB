Acerca de

¿Quién soy?

Soy estudiante del curso de Cultura Digital 2 en el Instituto Libertad. Este portafolio recopila todos los trabajos realizados durante el semestre, mostrando mi progreso y aprendizaje en diseño web, programación HTML, CSS y el uso de frameworks.

A través de este curso, he adquirido habilidades en:

  • Diseño de páginas web con HTML5 y CSS3
  • Aplicación de conceptos de teoría del color
  • Uso de frameworks responsive como Bootstrap
  • Creación de contenido digital con herramientas como Canva y Piktochart
  • Implementación de JavaScript para efectos interactivos
